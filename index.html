MLayout: A Multi-Source Dataset of 3D Indoor Layouts and Structured Descriptions for Text-to-3D Generation

Abstract

Recent progress in 3D generation has sparked growing interest in controllable scene synthesis from language. Layout quality significantly affects the structural coherence and semantic alignment in text-to-3D scene synthesis. However, existing datasets for indoor layouts are often limited in scale, diversity, or lack detailed structured annotations, which hinders robust learning and generalization.

To address these challenges, we present MLayout, a large-scale, multi-source dataset for 3D indoor layout generation from structured text. The dataset contains 16,929 layouts and over xxx object instances, collected from real-world scans, professional CAD designs, and procedurally generated scenes, each paired with detailed structured descriptions that cover global scene summaries, large furniture relations, and fine-grained item placements. This diverse and richly annotated resource supports learning robust spatial and semantic patterns across a wide range of indoor environments.

To evaluate the utility of MLayout, we train a text-conditioned diffusion model as a baseline for layout generation. Using this model, we conduct ablation studies showing that progressively incorporating the three data sources consistently improves generation quality, demonstrating their complementary nature. The model also achieves more realistic, semantically coherent, and controllable layouts compared to recent state-of-the-art layout generation methods based on large language models. We hope MLayout can serve as a useful foundation for future research in text-driven 3D scene synthesis.
